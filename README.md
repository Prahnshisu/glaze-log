# Glaze Kitchen Materials Log

A simple, mobile-friendly web tool for tracking glaze material weights and costs in a ceramics studio.

Designed to work anywhere with an internet connection, this log lets you add only the materials you actually use, automatically calculate costs, and export a clean PDF record.

---

## What this tool does

- Add materials **on demand** (no clutter from unused materials)
- Search materials quickly with type-ahead filtering
- Automatically calculates:
  - Subtotal = grams × price per gram
  - Total glaze cost
- Fixed material prices (to prevent accidental edits)
- Clear hazard icons for materials that require PPE
- Works on:
  - Desktop
  - Tablet
  - Phone
- Print or save as PDF once safety confirmation is acknowledged

---

## Workflow

1. Open the page in a browser
2. Enter:
   - Author’s name
   - Date
   - Glaze name
   - Firing type
3. Click **+** in the table header to add a material
4. Search for the material and select it
5. Enter the weight in grams
6. Repeat as needed
7. Confirm safety acknowledgement
8. Print or save as PDF

Only materials you add appear in the table.

---

## Safety notice

Users are required to confirm they understand studio safety responsibilities before exporting the record.

## Hosting

This tool is hosted using **GitHub Pages** and is available via a public URL.

To update it:
1. Edit the `index.html` file in this repository
2. Commit the changes
3. The live site updates automatically

---

## Notes

- This is a **static HTML file** (no server, no tracking, no data storage)
- All calculations run locally in the browser
- No data is collected or transmitted
